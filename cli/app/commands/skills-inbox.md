The user wants to check their Skills Exchange inbox.

Run this command and show the output:

```bash
skillshare inbox
```

The inbox can contain two types of items:

**Friend requests** — someone wants to be friends so they can exchange skills. For each one, show the accept and decline commands exactly as printed. The user must accept before either person can send skills to the other.

**Skills** — a slash command someone has sent. For each skill, explain that the user can install it by running `skillshare add <id>` or `/skills-add <id>` inside Claude Code.

If the inbox is empty, encourage the user to share their username so others can find and send them skills.
